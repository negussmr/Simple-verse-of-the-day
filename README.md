# ✝️ Simple Verse of the Day

A beautiful, minimalist web application that displays a daily Bible verse with a streak tracker. Built with a **liquid glass UI** and deep blue/black aesthetic—perfect for daily devotion.

![Verse of the Day Screenshot](https://via.placeholder.com/400x600/1a1a2e/FFFFFF?text=Verse+of+the+Day)

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 📖 **Daily Verse** | Shows a new Bible verse every day (365 verses included) |
| 🔥 **Streak Tracker** | Tracks how many consecutive days you've visited |
| 🎨 **Liquid Glass UI** | Beautiful frosted glass effect with animated background |
| 🎉 **Celebration Particles** | Colorful particles burst when your streak increases |
| 📤 **Share** | Share verses with friends via native share or clipboard |
| 📅 **Date Display** | Shows current date in a clean format |
| 📱 **Responsive** | Works perfectly on all devices (phone, tablet, desktop) |
| 🌐 **No API Required** | 100% self-contained—works offline |

- **LocalStorage** - Persistent streak tracking
